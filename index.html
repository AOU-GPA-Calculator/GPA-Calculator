How to Use

  • Add Course: Click "Add New Course" to add a course.
  • Enter Grade: Must be between 0 and 100.
  • Credit Hours: Must be a positive number.
  • Calculate: Click "Calculate GPA" to compute the cumulative GPA.
  • Merit: Merit classification appears next to the GPA.

كيفية الاستخدام

  • إضافة مقرر: اضغط على "Add New Course" لإضافة مقرر جديد.
  • إدخال الدرجة: يجب أن تكون الدرجة بين 0 و 100.
  • إدخال الساعات: يجب إدخال عدد ساعات موجب وصحيح.
  • حساب المعدل: اضغط "Calculate GPA" لحساب المعدل التراكمي.
  • التصنيف: يظهر التصنيف الأكاديمي بجانب المعدل.

About this Tool

This web-based GPA Calculator is developed as an academic project for the Arab Open University. It helps students quickly estimate their cumulative GPA on a 4.0 scale based on course grades (0–100) and credit hours. The tool is for guidance only and does not replace the official GPA calculation used by the university.

Features include input validation, GPA merit classification, dark/light mode, result printing, and automatic saving of the last entered courses using the browser's local storage.
